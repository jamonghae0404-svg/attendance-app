새로운 프로젝트를 시작합니다. 아래 요구사항을 바탕으로 전체 구조를 설계하고 단계별로 코딩을 진행해주세요. 모든 사용자 인터페이스(UI) 텍스트는 '한국어'로 작성해야 합니다.

1. 기술 스택:
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Icons: Lucide-react
- Features: PWA(Progressive Web App) 설정 포함

2. 디자인 가이드라인 (토스/카카오톡 스타일):
- 배경: 깨끗한 화이트 톤 (#FFFFFF)
- 포인트 컬러: 인디고 블루 (#4f46e5)
- 컴포넌트: 모서리 rounded-xl, 은은한 shadow-sm
- 모바일 최적화: 넉넉한 버튼 크기와 터치 피드백

3. 주요 기능 및 UI:
- 보안: 접속 시 '비밀번호 입력창(초기: 17120)' 카드 UI 노출
- 관리: 프로그램 생성 및 이용자 등록/종결(비활성화) 기능
- 출석부: [출석(초록), 결석(빨강), 조퇴(노랑)] 대형 버튼
- 사유 입력: 결석/조퇴 시 부드러운 아코디언으로 퀵 버튼(병원 방문, 가족 행사, 무단 결석) 노출
- 일지: 하단에 '오늘의 일지' 메모 섹션
- 주말 대응: 토/일요일 포함 모든 요일 선택 가능 (예: 360 어디나돌봄)

4. 통계 및 데이터:
- 대시보드: 주간(월~일) 및 월간 '실인원'과 '연인원' 통계 요약 카드
- 엑셀: 전체 출석 로그 및 요약 통계를 포함한 .xlsx 내보내기

5. 작업 순서:
- 먼저 프로젝트 폴더 구조와 데이터베이스(또는 로컬 스토리지) 설계 계획을 말해주세요.
